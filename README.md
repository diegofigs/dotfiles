# dotfiles

## How to install
```
Mac: 
 brew install yadm &&
 yadm clone https://github.com/diegofigs/dotfiles.git

Arch (fresh install):
 pacman -Syu sudo git --noconfirm &&
 sudo curl -fLo /bin/yadm https://github.com/TheLocehiliosan/yadm/raw/master/yadm && 
 sudo chmod a+x /bin/yadm &&
 yadm clone https://github.com/diegofigs/dotfiles.git

Arch (AUR with yay): 
 yay -Syu yadm-git &&
 yadm clone https://github.com/diegofigs/dotfiles.git
```
